# Projeto AVG – Veículo Autônomo com ESP32

##  Sobre o Projeto
Projeto acadêmico desenvolvido com foco em sistemas embarcados e integração entre hardware e software.

O veículo realiza monitoramento de temperatura e umidade, envia dados em tempo real para um servidor e permite controle via interface web.

##  Tecnologias Utilizadas
- ESP32
- Sensor DHT11
- Python
- Flask
- APIs REST

##  Funcionalidades
- Monitoramento de temperatura e umidade
- Envio de dados em tempo real
- Registro das informações em servidor
- Controle manual via interface web

##  Conceitos Aplicados
- Sistemas embarcados
- Comunicação via APIs
- Integração cliente-servidor
- Desenvolvimento backend
